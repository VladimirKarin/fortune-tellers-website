# Fortune Teller Website

A simple website for a fortune teller in RU & LT languages. Written as an exam project.

## Features

-   Responsive design
-   Carousel for displaying images
-   Countdown to the next moon phase
-   Calendar for displaying moon phases
-   Mobile navigation functionality

## Technologies Used

-   HTML5
-   CSS3 (including Flexbox and Grid)
-   JavaScript (including React, MUI, and React Hooks)
-   Ionicons for icons
-   Google Fonts for typography

## Future Plans

-   Add more functionality to the countdown
-   Add more details to the calendar
-   Add more images to the carousel
-   Make the website more interactive
